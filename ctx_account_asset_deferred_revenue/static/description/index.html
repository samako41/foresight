Manage Your Fixed Assets and Deferred Revenue & Expenses

This module allows you to manage your fixed asset and deferred revenue and expense in an easy way.

Overview

This module helps you to create assets and deferred revenues/expenses and automate the calculation and posting to accounting on scheduled period.

Key Features
Ability to set the already depreciated value to be excluded from Odoo calculation, In case of migration from another software
Ability to create assets types.
Ability to create asset manually or automatically from vendor bill.
Ability to sell and dispose assets in an easy way.
Ability to open assets from vendor bill, In case of asset creation from vendor bill.
Ability to create deferred revenue/expense types.
Ability to create (draft or automatically running depend on configuration in deferred revenue/expense type) deferred revenue/expense automatically from customer invoice / vendor bill.
Ability to create deferred revenue/expense manually.
